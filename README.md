## Psy

Агент для работы с психологическим состоянием

### Навыки

- **dbt-fact-checking** — Психотерапевтический агент на основе диалектической поведенческой терапии (DBT) с техникой "Проверка фактов". Помогает анализировать эмоции, выявлять когнитивные искажения и определять, соответствуют ли эмоциональные реакции фактам ситуации.

- **procrastination-coach** — Психологическая поддержка при прокрастинации и перфекционизме. Помогает преодолевать внутренние барьеры, выявлять психологические паттерны и создавать детальные планы действий.

## Dev

Агент для разработки ПО с системой **Superpowers**

### Описание

**Superpowers** — система workflow для разработки ПО, используемая агентами-кодировщиками (Claude Code, OpenCode, Codex, Cursor). Построена на наборе компонуемых "скиллов" и начальных инструкций.

### Возможности

- Автоматизация TDD (Test-Driven Development)
- Систематическое отладка (debugging)
- Code review и рефакторинг
- Планирование и управление задачами
- Brainstorming и создание дизайна

### Основные Workflow

| Триггер | Скилл | Действие |
|---------|-------|----------|
| Начало разработки | brainstorming | Уточнить требования, создать дизайн |
| Есть дизайн | writing-plans | Создать план реализации |
| Есть план | subagent-driven-development | Запустить субагентов |
| Реализация | test-driven-development | RED-GREEN-REFACTOR цикл |
| Между задачами | requesting-code-review | Ревью кода |
| Завершение | finishing-a-development-branch | Merge/PR/cleanup |

### Установка

```bash
# OpenCode
/opencode plugin install superpowers

# Обновление
/opencode plugin update superpowers
```

## Life

Агент для управления задачами, проектами, привычками и заметками

### Инструменты

- **Singularity MCP Server** — Работает с API Singularity для управления проектами, задачами, записными книжками, привычками и заметками
- **Singularity Backup Server** — Извлечение данных из JSON бэкапов Singularity с возможностью фильтрации и экспорта

### Использование

```bash
# Создание задач, проектов и других операций
# Используйте инструменты MCP сервера singularity-task

# Извлечение данных из бэкапа
python3 scripts/extract_singularity_data.py --status incomplete --no-include-basket
```
