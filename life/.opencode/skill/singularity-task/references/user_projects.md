# Справочник проектов пользователя

Этот файл содержит личные проекты пользователя в Singularity. Используйте этот справочник, когда пользователь просит добавить задачи, создать элементы или упоминает действия, которые должны быть отнесены к определенным проектам.

## Список проектов

### Agile Life
**Назначение:** Задачи, которые обслуживают циклы спринт-планирования
**Использовать когда:** Пользователь упоминает спринты, циклы планирования, agile методологию, ретроспективы

### Идеи
**Назначение:** Хранилище идей
**Использовать когда:** Пользователь говорит "у меня есть идея", "сохрани эту идею", мозговые штурмы

### Лекарства
**Назначение:** Задачи по приему лекарств
**Использовать когда:** Пользователь упоминает прием лекарств, рецепты, таблетки, график приема медикаментов

### Самовыражение и творчество
**Назначение:** Творческие задачи и самовыражение
**Использовать когда:** Пользователь упоминает искусство, письмо, музыку, творческие проекты, деятельность по самовыражению

### Расходы
**Назначение:** Запланированные расходы
**Использовать когда:** Пользователь упоминает запланированные покупки, бюджетные расходы, будущие траты

### ДР
**Назначение:** Дни рождения, годовщины смерти различных людей
**Использовать когда:** Пользователь упоминает дни рождения, годовщины, памятные даты

### Образ жизни и среда
**Назначение:** Задачи по улучшению домашнего окружения
**Использовать когда:** Пользователь упоминает улучшение дома, организацию пространства, оптимизацию окружения

### Эмоц-псих
**Назначение:** Задачи по эмоциональному и психологическому здоровью
**Использовать когда:** Пользователь упоминает терапию, ментальное здоровье, эмоциональное благополучие, психологические упражнения

### Тело и Здоровье
**Назначение:** Задачи, связанные с телом и здоровьем, физические упражнения
**Использовать когда:** Пользователь упоминает тренировки, упражнения, физическое здоровье, визиты к врачу, отслеживание здоровья

### Дом
**Назначение:** Домашние задачи - ремонт бытовой техники, уборка и т.д.
**Использовать когда:** Пользователь упоминает уборку, ремонт, домашние дела, обслуживание дома

### work
**Назначение:** Задачи по основной работе в 2ГИС
**Использовать когда:** Пользователь явно говорит "рабочая задача" или упоминает 2ГИС
**Примечание:** Проект по умолчанию для рабочих задач

### work franch
**Назначение:** Вторая работа - франчайзинг Смоленск и Вологда
**Использовать когда:** Пользователь упоминает франшизу, Смоленск, Вологду, задачи, связанные с франчайзингом

### Avito
**Назначение:** Задачи, связанные с продажей вещей
**Использовать когда:** Пользователь упоминает продажу вещей, объявления на Avito, размещение рекламы

### Купить домой
**Назначение:** Задачи по покупке товаров для дома и продуктов
**Использовать когда:** Пользователь упоминает покупку продуктов, шопинг для дома, покупки для дома

### Еда
**Назначение:** Задачи по приготовлению пищи, рецепты, планирование меню
**Использовать когда:** Пользователь упоминает приготовление еды, рецепты, меню, продукты для готовки, кулинария

## Руководство по использованию

1. **Когда пользователь упоминает задачу без указания проекта:** Проанализируйте содержание задачи и назначьте наиболее подходящий проект из этого списка

2. **Когда пользователь говорит "добавь рабочую задачу":** Всегда добавляйте в проект "work"

3. **Когда контекст неоднозначен:** Спросите у пользователя, какой проект он предпочитает, предложив 2-3 наиболее подходящих варианта

4. **При создании новых похожих задач:** Проверьте, существуют ли похожие задачи в определенных проектах, и поддерживайте согласованность

5. **Названия проектов на русском:** Используйте точные русские названия, как указано выше, при ссылке на проекты
