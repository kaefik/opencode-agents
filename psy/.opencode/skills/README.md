# DBT Fact-Checking Agent для opencode.ai

Психотерапевтический агент на основе диалектической поведенческой терапии (DBT) с техникой "Проверка фактов".

## Быстрый старт

1. Загрузите файл `dbt-fact-checking.skill` в opencode.ai
2. Начните диалог с запроса типа: *"Помоги мне разобраться с моими чувствами по поводу..."*
3. Следуйте структурированному процессу из 6 шагов

## Что делает этот навык

Агент помогает:
- ✅ Объективно анализировать эмоциональные ситуации
- ✅ Отделять факты от интерпретаций
- ✅ Выявлять когнитивные искажения
- ✅ Оценивать соответствие эмоций реальности
- ✅ Принимать взвешенные решения

## 6 шагов процесса

1. **Описать ситуацию** - объективные факты без суждений
2. **Назвать эмоцию** - идентификация и оценка по шкале 0-10
3. **Выявить мысли** - интерпретации и когнитивные искажения
4. **Проверить доказательства** - анализ за и против
5. **Переоценить эмоцию** - соответствует ли она фактам?
6. **Выбрать действие** - что делать дальше

## Примеры запросов

```
Я очень злюсь на коллегу, но не знаю, оправдана ли моя злость

Помоги мне разобраться с тревогой о предстоящей встрече

Мне нужна помощь, чтобы понять свои чувства в этой ситуации
```

## Ключевые особенности

- 🎯 Структурированный подход из 6 шагов
- 💭 Сократический метод (вопросы, а не ответы)
- 🤝 Тёплая терапевтическая поддержка
- 🔍 Выявление когнитивных искажений
- 📊 Количественная оценка эмоций

## Что включено

- `dbt-fact-checking.skill` - основной файл навыка
- Справочник по когнитивным искажениям
- Коллекция терапевтических вопросов
