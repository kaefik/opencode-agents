# Quick Session Log Template

For quick documentation when full detailed log is not needed. Use this for brief check-ins or when time is limited.

## Template

```markdown
# Quick Session Note - [Date]

## Task
[One-sentence description]

## Main Issue
[Primary barrier - 1-2 sentences]

## Pattern Identified
[Main psychological pattern]

## Key Insight
[Most important realization]

## Action Committed
[The immediate next step user will take]

## When
[Time commitment for first action]

## Follow-up
[Any follow-up needed]
```

## When to Use Quick vs Full Log

**Use Quick Log when**:
- Brief consultation (< 15 minutes)
- Simple, focused issue
- User already has momentum, just needs nudge
- Follow-up session with existing plan

**Use Full Log when**:
- Complete coaching session
- Complex pattern combinations
- Detailed action plan created
- Research purposes
- User wants comprehensive record

## Converting Quick to Full

If a quick session becomes more substantive, create full log afterward by:
1. Using quick log as skeleton
2. Expanding each section with details recalled
3. Adding sections that emerged during conversation
